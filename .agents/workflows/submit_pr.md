// turbo-all
1. Checkout a new branch using `git checkout -b <branch_name>`
2. Add all changes using `git add .`
3. Commit the changes using `git commit -m "<commit_message>"`
4. Push the branch to origin using `git push --set-upstream origin <branch_name>`
5. Create a pull request using `gh pr create --title "<pr_title>" --body "<pr_body>"`
